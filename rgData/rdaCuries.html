Data using the RDA vocabularies

Curie prefixes

These prefixes are recommended for use in a compact URI (curie), and are used in RDA Registry documentation.

prefix namespace element set
rdac: http://rdaregistry.info/Elements/c/ RDA Classes
rdaa: http://rdaregistry.info/Elements/a/ Agent properties
rdaad: http://rdaregistry.info/Elements/a/datatype/ Agent datatype properties
rdaao: http://rdaregistry.info/Elements/a/object/ Agent object properties
rdae: http://rdaregistry.info/Elements/e/ Expression properties
rdaed: http://rdaregistry.info/Elements/e/datatype/ Expression datatype properties
rdaeo: http://rdaregistry.info/Elements/e/object/ Expression object properties
rdai: http://rdaregistry.info/Elements/i/ Item properties
rdaid: http://rdaregistry.info/Elements/i/datatype/ Item datatype properties
rdaio: http://rdaregistry.info/Elements/i/object/ Item object properties
rdam: http://rdaregistry.info/Elements/m/ Manifestation properties
rdamd: http://rdaregistry.info/Elements/m/datatype/ Manifestation datatype properties
rdamo: http://rdaregistry.info/Elements/m/object/ Manifestation object properties
rdan: http://rdaregistry.info/Elements/n/ Nomen properties
rdand: http://rdaregistry.info/Elements/n/datatype/ Nomen datatype properties
rdano: http://rdaregistry.info/Elements/n/object/ Nomen object properties
rdap: http://rdaregistry.info/Elements/p/ Place properties
rdapd: http://rdaregistry.info/Elements/p/datatype/ Place datatype properties
rdapo: http://rdaregistry.info/Elements/p/object/ Place object properties
rdat: http://rdaregistry.info/Elements/t/ Time-span properties
rdatd: http://rdaregistry.info/Elements/t/datatype/ Time-span datatype properties
rdato: http://rdaregistry.info/Elements/t/object/ Time-span object properties
rdau: http://rdaregistry.info/Elements/u/ Unconstrained properties
rdaw: http://rdaregistry.info/Elements/w/ Work properties
rdawd: http://rdaregistry.info/Elements/w/datatype/ Work datatype properties
rdawo: http://rdaregistry.info/Elements/w/object/ Work object properties
rdax: http://rdaregistry.info/Elements/x/ RDA Entity properties
rdaxd: http://rdaregistry.info/Elements/x/datatype/ RDA Entity datatype properties
rdaxo: http://rdaregistry.info/Elements/x/object/ RDA Entity object properties
rdaz: http://rdaregistry.info/Elements/z/ Meta-element properties

The prefix is suggested on the Registry webpage for each element set.
