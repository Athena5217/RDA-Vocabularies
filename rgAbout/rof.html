About the RDA Registry and vocabularies

RDA/ONIX Framework

The RDA/ONIX Framework (ROF) is an RDF representation of the RDA/ONIX Framework for Resource Categorization developed for RDA.

The Framework is the basis of the RDA Carrier Type, RDA Content Type, and RDA Media Type value vocabularies.

Element set

The element set contains 9 classes for combinations of attributes used to construct resource categories, and 24 properties for relating categories to attribute values.
