Export files for translation

The download links to the beta-test Export tab of the OMR value vocabulary.

Anyone can export the current data by using the OMR export facility. Import is restricted to authorized OMR accounts.

  • Use the Create New Export option to export the file for translation.
  • Choose the Select only in use option to select the data columns to export. Do not amend the selection of individual columns.
  • Use the Advanced option:
    • Exclude Deprecated: Yes (select)
    • Additional Language: Select the language of translation from the list.
      • Contact the RSC Translations Working Group if the language is missing.
    • Do not select any other options.
  • Use the Export CSV option to export the file.
  • Save the export file.
